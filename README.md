\# Akces Booth



Polska alternatywa dla ChackTok - aplikacja do sterowania fotobudka 360.



\## Architektura



\- \*\*recorder/\*\* - Flutter app na OnePlus 13 (nagrywanie + motor control)

\- \*\*station/\*\* - Flutter app na Tab A11+ (stacja dla gosci, QR delivery)

\- \*\*backend/\*\* - Flask API na Raspberry Pi (hosting filmow, admin panel)

\- \*\*docs/\*\* - Dokumentacja projektu
